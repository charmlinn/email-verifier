验证参数

参数说明

开启 SMTP 验证
连接邮箱域名的 MX 服务器，尝试用 SMTP RCPT 判断邮箱是否可投递；不发送真实邮件。
检测 catch-all
先探测一个随机邮箱。如果域名接收任意地址，具体账号是否存在就无法可靠确认。
域名拼写建议
对 gmai.com 这类疑似拼写错误给出建议域名。
检测 Gravatar
查询邮箱是否有公开 Gravatar 头像，可作为账号活跃线索，不代表邮箱一定可投递。
Yahoo API 验证
仅对 Yahoo 邮箱相关 MX 生效，会模拟访问 Yahoo 注册页检查用户名是否已存在；它不是官方接口，可能受页面结构、限流或风控影响。

映射结果说明

decisionCode / decisionText
最终业务决策：可接入、建议人工复核、不建议接入或验证异常。
mailboxProfile
邮箱画像，包括企业域名/公共邮箱、角色邮箱/个人邮箱、一次性邮箱风险。
deliverability
投递判断，包括是否执行 SMTP、是否确认可收信、是否存在 catch-all 风险。
evidence
每一条判断依据，方便定位为什么被拒绝或为什么需要复核。
